# RentEase

RentEase is a rental storefront web app for furniture and appliances. Users can build a rental bundle, compare monthly costs, select a duration, and submit a request through a clean dashboard.

## Files

- `index.html` — main application page
- `styles.css` — styles and layout
- `app.js` — catalog logic, state, and interactions
- `main.js` — Electron desktop wrapper
- `package.json` — project metadata and Electron scripts
- `run-rentease.bat` — desktop launch helper for Windows
- `rentease.ico` — custom shortcut icon

## How to run locally

1. Install Node.js: https://nodejs.org/
2. Open PowerShell in the project folder
3. Run `npm install`
4. Run `npm start`

## Project features

- product catalog with search, filtering, and sorting
- rental plan duration and location controls
- bundle summary with monthly rent and deposit calculation
- checkout request form
- desktop shortcut support

## GitHub deployment

To publish this project on GitHub and get links, install Git and push the repo to GitHub. To deploy as a live website, use GitHub Pages from the repository settings.
